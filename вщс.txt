1. Создали папку, в ней открыли терминал. 
Команда git init - создание локального репозитория
2. Зашли на свой гит хаб, создали там удаленный репозиторий, по кнопке CODE<> нашли ссылку на репозиторий
3. Вернулись в терминал и ввели команду: 
git clone cсылка на репозиторий
4. пошли писать код самый крутой на свете
5. зашли в папку, где лежат все наши файлы
git status - проверили статус файлов
6. git add . - забросили все на локальный репозиторий 
7. git commit -m "комментарий" - прокомментировали изменения, внесенные в файлы
8. git push origin main - закинули все на удаленный репозиторий
9. зашли в git hub  и радуемся что файлы теперь там
